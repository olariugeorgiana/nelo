Intro

NeLo is a web application whose purpose is the management of a hotel reservations system in Constanta .

The client can choose between the type of room and other options offered like air conditioning or internet.

In the admin area the hotel employees can add new reservation or registration and the administrator of the web application can change the content.

NeLo

Used Technologies
The app was developed using MVC Pattern. In the front end of the web app we used valid HTML5 and CSS, Javascript, Jquery and Spring tiles.

NeLo

Used Technologies
For the back end of our web app we used Java Enterprise Edition with Spring Framework and Hibernate for PostgreSQL database.

The Web Module
View rooms, facilities, Info, User Account
Once the user is logged in, he see details about rooms, facilities and other hotel information by clicking menu buttons: "Rooms", "Facilities", "Info", "Home". Also the is a section "Account" in which are shown details about user activity(reservation and registrations).

NeLo

The Web Module
Book a room
The client may choose the period, and between one or more facilities.
After he choose an available room, he can pick from 3 types of meals: "Breakfast ", "Lunch ","Dinner".
The price for meals and room will automatically be computed and shown.

The Hotel Admin Module
User Role - Add a user
In the admin module two user roles have access: "Receptionist" and "Administrator". This users can be created by the master admin of the application. After creating one user account a mail will be sent to his address with the credentials.

NeLo

The Hotel Admin Module
User Role - Add a reservation - Add a registration
The "Receptionist" can save new reservations or registrations. Also he can see a list of all the reservations made by the clients from the web site using the "Book now" form.
He can "Edit", "Delete" or change a reservation to a registration.

The Hotel Admin Module
Add a room
The administrator can add a new room. He must fill a form with data about new room: "Room number", "Room Type", "Description", "Pictures", "Price". After saving the room, it will appear in the web application .
